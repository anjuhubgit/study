command line utilities
In the final expansion of Kernel-Land, Script realized that he didn't have to do everything by hand. The kingdom was filled with Specialist Guilds—master craftsmen who could manipulate text and files with incredible speed.Instead of writing long functions, Script learned to "hire" these specialists to process his data.
1. The Scout: find and grepWhen Script needs to locate items in the vast wilderness of the kingdom, he calls the Scouts.find: This scout specializes in finding files based on their name, size, or age.“Find me every scroll that ends in .txt!” → find /kingdom -name "*.txt"grep: Once a file is found, grep looks inside to find specific words.“Find the word 'Dragon' inside this scroll!” → grep "Dragon" scroll.txt
2. The Surgeon: cut and sedSometimes a message is too long, or it contains mistakes that need a delicate touch.cut: This specialist acts like a pair of scissors, snipping out only the columns of data Script needs.“Give me only the first name from this comma-separated list!” → cut -d',' -f1 names.csvsed (Stream Editor): The master of disguise. sed can find a word and replace it with another as the text flows by.“Change every 'Goblin' to 'Friend'!” → sed 's/Goblin/Friend/g' report.txt
3. The Librarian: sort and uniqWhen Script is overwhelmed by a messy pile of reports, he visits the Librarians to bring order to the chaos.sort: This specialist lines everything up in alphabetical or numerical order.uniq: Once sorted, uniq removes any duplicate lines, so Script only sees unique entries.“Give me a sorted list of unique visitors!” → cat guest_list.txt | sort | uniq
4. The Architect: awkThe most powerful specialist of all is awk. While cut is a pair of scissors, awk is a full construction crew. It treats every line like a row in a spreadsheet and can even perform math.If Script has a list of treasures and their gold values, awk can sum them up instantly:cat inventory.txt | awk '{sum += $2} END {print "Total Gold:", sum}'The Great Collaboration (The "One-Liner")The true magic happens when Script uses a Pipe (|) to make these specialists work together in a single line.Imagine Script needs to find the top 3 most common "Warning" messages in a massive log file:Bash# The Collaboration:
grep "WARNING" kingdom.log | cut -d':' -f2 | sort | uniq -c | sort -nr | head -n 3
grep grabs only the warnings.
cut snips out the specific error message.
sort groups identical messages together.
uniq -c counts how many times each message appeared.
sort -nr sorts those counts from highest to lowest.
head -n 3 takes only the top 3 results.
